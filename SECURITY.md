# Kebijakan Keamanan (Security Policy)

Kami di **UTDI Hackathon Club** berkomitmen untuk menyediakan proyek web yang aman bagi pengguna dan pengembang. Meskipun proyek ini adalah proyek pembelajaran mahasiswa, kami tetap menghargai pelaporan masalah keamanan secara bijak.

---

## Versi yang Didukung (Supported Versions)

Kebijakan keamanan ini hanya berlaku untuk versi rilis aktif berikut:

| Versi | Didukung |
| ------- | --------- |
| v1.2.x  | Ya        |
| v1.1.x  | Tidak     |
| < v1.0  | Tidak     |

---

## Melaporkan Kerentanan (Reporting a Vulnerability)

Harap **jangan** melaporkan celah keamanan publik melalui sistem GitHub Issues.

Jika Anda menemukan kerentanan keamanan pada kode proyek ini (misalnya celah XSS, kebocoran data simulasi, atau masalah dependensi eksternal), silakan kirimkan laporan secara langsung ke:
* **Email Pengurus:** [Email Pengurus/Admin Club]
* **WhatsApp Admin:** [Kontak Admin Club]

### Proses Penanganan Laporan:
1. Kami akan mengonfirmasi laporan Anda dalam waktu **48 jam**.
2. Tim pengembang klub akan menganalisis kerentanan secara internal dan menyiapkan perbaikan kode.
3. Perbaikan celah keamanan akan dirilis ke branch utama GitHub secepatnya, dan Anda akan dicatat sebagai kontributor pelapor celah keamanan di bagian Changelog (jika Anda berkenan).

Terima kasih telah ikut membantu menjaga keamanan proyek ini! 🛡️💚
